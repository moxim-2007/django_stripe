# Run

### 1. Сreate a file `.env` in the root of the project

### 2. Сreate variables and assign values

   ```
   STRIPE_API_PUBLIC_KEY
   STRIPE_API_SECRET_KEY
   ```

### 3. Run

`docker-compose up --build`

### 4. Open http://127.0.0.1:8000/


#### superuser configuration:

   ```
   USERNAME: admin
   PASSWORD: admin
   ```
